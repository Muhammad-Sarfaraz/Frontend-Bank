- Project Setup

```sh
npm install
```

- Compile and Hot-Reload for Development

```sh
npm run dev
```

- Compile and Minify for Production

```sh
npm run build
```

#### Endpoint
```
http://127.0.0.1:8000/api
```

### CORS
In case of CORS issue, set the frontend url in header of Cors middleware of the project.
